# QA Testing Strategy

## Static Tests

- Route package validity.
- Multi-subject route fixtures.
- Source hygiene.
- UI completeness markers.
- Storage migration and contract checks.

## Runtime Tests

- Backend health and key API endpoints.
- Web dashboard and workflow navigation.
- Electron shell startup.

## Browser Targets

Check desktop width, tablet width, and mobile width for text overflow, navigation reachability, and workflow continuity.

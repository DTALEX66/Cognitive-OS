# Project Spec Index

Lumina Knowledge Base is a local-first learning route workspace. The product surface is split into Web UI and Desktop EXE UI, both backed by the same route package, review, memory palace, training, sync, and report flows.

Core specs:

- Product requirements: `docs/product/PRODUCT_REQUIREMENTS_SPEC.md`
- Roadmap: `docs/product/ROADMAP_COMMERCIALIZATION_PLAN.md`
- UI system: `docs/design/UI_DESIGN_SYSTEM.md`
- Brand execution: `docs/brand/BRAND_EXECUTION_GUIDELINES.md`
- Engineering quality: `docs/engineering/CODING_STANDARDS.md`
- Release gate: `docs/release/COMMERCIAL_RELEASE_GATE.md`

Every feature must have real data, a visible user path, and at least one validation route through `npm run test` or a targeted script.

# Product Requirements Spec

## Goal

Provide a personal knowledge system that turns documents and concepts into sourced learning routes, memory palace anchors, active recall sessions, and local reports.

## Primary Workflows

- Import or create knowledge.
- Generate or edit a route package.
- Review facts and memory encodings.
- Train with active recall.
- Inspect mastery and weak nodes.
- Sync or export local assets.

## Acceptance Criteria

- All primary pages load without placeholder data.
- Route packages contain nodes, source text, recall prompts, and visual mappings.
- User changes are saved locally before being shared or exported.
- Desktop EXE opens to a focused workspace, not a marketing landing page.

# Browser Smoke Test Plan

## Goal

v0.7 adds an HTTP-level browser smoke gate for all static routes generated by the Next.js app.

## Command

```bash
npm run build
npm run smoke:pages
```

## Routes covered

```text
/
/upload
/review
/palace
/training
/report
/routes
/community
/settings
```

## What this catches

- Pages that fail at runtime after a successful TypeScript build.
- Missing static routes.
- Bad app-shell rendering.
- Accidental removal of the `Knowledge-Base` brand marker.

## What this does not replace

This is not a visual regression test. Future v0.8 should add Playwright screenshot baselines for desktop and mobile viewports.

# Lumina AI 项目压缩上下文

## 1. 项目定位

Lumina AI 是一个 AI 学习路线平台，不是普通 AI 总结器，也不是让 AI 替用户学习，而是 AI 帮用户把资料转化为准确、可记忆、可训练、可复用的学习路线。

核心一句话：把知识变成路线，把路线变成资产，把资产变成社区。

长期基础：本地优先 + 云端增强 + 知识严谨 + 记忆稳定 + 社区共建。

## 2. 用户流程

资料导入 → AI 解析知识 → 事实层确认 → AI 记忆编码 → 空间路线/记忆宫殿 → 主动回忆训练 → 学习报告 → 本地资产沉淀 → 社区路线分享。

## 3. 五大引擎

1. Knowledge Integrity Engine / 知识严谨性引擎
2. Memory Encoding Engine / AI 记忆编码引擎
3. Local Memory Asset Engine / 本地记忆资产引擎
4. Learning Verification Engine / 学习验证引擎
5. Community Route Engine / 社区学习路线引擎

## 4. 事实层与记忆层

事实层：时间、人物、定义、公式、条文、标准答案、适用条件、原文依据。

记忆层：图像、隐喻、故事、空间位置、口诀、联想、回忆提示。

规则：事实层必须准确；记忆层可以夸张，但必须解释映射关系；记忆可以诗意化，事实不能漂移。

## 5. 本地资产原则

AI 生成一次，用户确认一次，本地长期复用。

路线生命周期：draft / confirmed / frozen / revised。

confirmed/frozen 后不得自动重新生成记忆图和路线。重生成只能生成候选版本，用户确认后才替换当前版本，旧版本必须保留。

## 6. 社区路线原则

社区不是普通分享，而是带版本、带来源、带校验、带提醒、可调用、可分叉、可追溯的学习路线分享。

社区路线四层：事实层、教学层、记忆层、视觉层。

改动风险标签：A 视觉改动，B 记忆表达改动，C 学习结构改动，D 事实层改动。D 类必须强提示。

## 7. 品牌与 UI

主品牌：Lumina AI。
中文意象：琉明知途。

导航保持清晰：首页 / 路线 / 社区 / 报告 / 设置。
页面标题可诗意化：知途星台、文稿析星、忆象织路、印忆殿、知境试炼、星途纪略、共建星河、知衡门禁等。

右上角双语切换位于搜索框右侧、通知铃铛左侧，显示：🌐 中 / EN。

UI 风格：现代 SaaS、白色/浅灰底、蓝紫渐变光感、圆角卡片、轻阴影、清晰导航、诗意但不玄、学习型产品可信感。

## 8. 工程质量门禁

跨 ChatGPT 网页版 / Codex 桌面版 / VS Code / Cursor / 其他 IDE 切换前必须跑：

```bash
npm run quality
npm run smoke:pages
npm run prehandoff
npm run audit:high
git diff --check
```

目标是防止中文乱码、UTF-8/BOM、CRLF/LF、语法错误、类型错误、i18n key 不一致、资源路径错误、RoutePackage Schema 漂移、冻结资产被误覆盖。

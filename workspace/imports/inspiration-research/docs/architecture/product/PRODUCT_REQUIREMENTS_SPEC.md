# 产品需求规范 PRD

## 1. 产品定义

Lumina AI 是一个本地优先、云端增强、知识严谨、记忆稳定、社区共建的 AI 学习路线平台。它帮助用户把资料转化为可追溯的知识结构、可解释的记忆图像、稳定的空间路线和可验证的训练任务。

## 2. 目标用户

| 用户 | 核心需求 |
|---|---|
| 学生 / 备考用户 | 把难背资料变成可记忆、可复习、可检测的路线 |
| 自学者 | 管理长期学习资产，不反复重建知识结构 |
| 教师 / 内容作者 | 创建可分享、可复用、可审核的学习路线模板 |
| 机构 | 建立私有课程路线库和学习数据闭环 |

## 3. 当前 MVP 必须完成

```text
/routes      路线资产工作台
/upload      资料导入与解析演示
/review      事实层与记忆层确认
/palace      稳定记忆路线展示与冻结
/training    主动回忆训练与本地记录
/report      基于真实训练记录的学习报告
/community   社区路线信任模型展示
/settings    双语与质量门禁设置
```

## 4. 当前不做

```text
真实 AI API
真实支付
真实登录系统
完整社区后端
复杂 3D 游戏化宫殿
医疗诊断 / 法律结论 / 投资建议
自动覆盖用户已冻结路线
```

## 5. 核心成功标准

1. 用户能理解产品不是 AI 总结器，而是学习路线平台。
2. 用户能完成从路线载入、节点确认、路线冻结、训练、报告的闭环。
3. 用户能感知知识严谨性、记忆编码、本地资产、学习验证、社区信任五大能力。
4. 页面 UI 有统一品牌气质，不像临时 AI 生成页面。
5. 代码与文档可被 Codex 接续开发。

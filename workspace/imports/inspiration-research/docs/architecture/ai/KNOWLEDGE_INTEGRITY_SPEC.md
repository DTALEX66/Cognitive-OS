# 知识严谨性规范

## 1. 事实层与记忆层

事实层：时间、人物、定义、公式、条文、标准答案、原文依据。
记忆层：图像、隐喻、故事、口诀、空间节点。

规则：记忆可以抽象，但事实不能漂移。

## 2. 来源追溯

正式知识点必须有：
- sourceText
- sourceLocation
- sourceType
- coreFact
- riskLevel

无来源内容只能标记为：AI 推测 / 需要确认 / 辅助解释。

## 3. 风险分级

| 等级 | 类型 | 规则 |
|---|---|---|
| 1 | 普通常识、普通历史事实 | 可自动生成，保留来源 |
| 2 | 考试、数学、科学、编程 | 需条件和错误检查 |
| 3 | 医学、法律、金融、心理 | 必须显示风险提示 |
| 4 | 诊断、法律结论、投资决策 | 禁止自动定论 |

## 4. 通过规则

```text
Accuracy < 0.95：不得进入正式路线
Source Fidelity < 0.90：必须提示确认
Risk Level >= 3：必须显示风险说明
Fact Layer changed：社区分享强提示
No sourceText：不得标记已校验
```

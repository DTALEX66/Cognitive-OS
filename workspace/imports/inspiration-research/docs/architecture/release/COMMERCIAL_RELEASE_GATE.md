# 商用发布门禁

## 1. 当前 v0.5 状态

本包是商用前端静态 MVP，不等于完整生产系统。可以作为：
- UI 交付参考。
- Codex 开发起点。
- 投资/团队演示原型。
- 产品规范基线。

## 2. 商用前必须补齐

```text
真实依赖安装和构建通过
页面响应式适配
真实错误处理
IndexedDB 替代核心 localStorage
图片资产版本存储
社区后端权限模型
隐私政策与用户协议
版权审核流程
AI Provider 安全边界
真实测试覆盖
```

## 3. 不可商用发布条件

- `npm run build` 未通过。
- i18n key 不对齐。
- RoutePackage 校验失败。
- 冻结资产仍会被自动覆盖。
- 高风险知识无提醒。
- 社区路线无改动提示。
- 包含真实隐私或 API Key。

## 4. 最小商用 Beta 条件

- 首页、路线、训练、报告体验稳定。
- 本地导入导出可靠。
- 双语可用。
- 视觉达到统一设计规范。
- 有用户协议、隐私政策、版权提醒。
- 发布前完整跑过 lint/typecheck/test/build。

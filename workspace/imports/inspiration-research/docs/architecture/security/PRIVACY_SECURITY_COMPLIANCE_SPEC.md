# 隐私、安全与合规规范

## 1. 本地优先

默认学习资产保存在本地。云端只作为增强能力，不作为基础依赖。

## 2. 隐私边界

社区分享默认只分享路线模板，不分享：
- 个人训练记录
- 私人笔记
- 错题历史
- 复习计划
- 上传原文全文
- 个人身份信息

## 3. 版权提醒

公开分享路线前必须提醒用户确认拥有分享权限。教材、课程、付费资料、书籍原文不能直接大段公开。

## 4. 高风险内容

医学、法律、金融、心理健康、工程安全等内容必须显示学习用途与风险边界。

## 5. 安全发布前检查

- 不含 API Key。
- 不含私密文件。
- 不含用户真实数据。
- 不含未经授权的大段版权内容。
- 不含危险操作指导。

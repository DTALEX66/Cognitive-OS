# 社区路线信任规范

## 1. 社区定位

社区不是图片分享，而是可信学习路线资产分享。

## 2. 路线四层结构

| 层级 | 内容 | 改动风险 |
|---|---|---|
| 事实层 | 时间、定义、公式、条文、答案、来源 | 最高 |
| 教学层 | 释义、讲解、训练题、复习建议 | 中高 |
| 记忆层 | 图像描述、故事、口诀、空间节点 | 中 |
| 视觉层 | 图片、风格、颜色、构图 | 低 |

## 3. 调用前确认

用户复制社区路线前必须展示：
- 作者
- 来源路线
- 审核状态
- 事实层改动数量
- 记忆层改动数量
- 图片层改动数量
- 适用对象
- 风险提示

## 4. 改动风险标签

A：视觉改动。
B：记忆表达改动。
C：学习结构改动。
D：事实层改动，必须强提醒。

## 5. 审核等级

```text
Level 0 未审核
Level 1 AI 自动校验
Level 2 社区稳定
Level 3 教师审核
Level 4 官方推荐
```

## 6. 排序规则

社区排序不能只看热度，必须综合：
- 来源完整度
- 审核等级
- 事实错误举报率
- 完成学习率
- 回忆正确率
- 纠错通过率

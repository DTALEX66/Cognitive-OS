## 视觉原则

- 顶部右侧必须保留双语切换：`中 / EN`。
- 顶层导航保持清晰，不要过度诗意化。
- 页面主标题和主题卡片可以诗意化。
- 操作按钮必须保持清晰可理解。
- 所有高风险、事实层改动提示必须直接明确。

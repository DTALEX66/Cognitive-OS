# 品牌执行规范

## 1. 品牌结构

| 类型 | 建议 |
|---|---|
| 主品牌 | Lumina AI |
| 中文意象名 | 琉明知途 |
| 产品一句话 | 把知识变成路线，把路线变成资产 |
| 气质 | 光、迹、纹、途、星、殿、境 |

## 2. 命名原则

1. 顶层导航保持直白，不牺牲可用性。
2. 页面主标题可以诗意化。
3. 按钮文案必须清晰，不能让用户猜。
4. 技术能力名可以品牌化，但旁边要有普通解释。
5. 社区和风险提示必须严谨，不要过度诗意。

## 3. 推荐命名体系

| 功能 | 诗意名 | 普通解释 |
|---|---|---|
| 首页 | 知途星台 | 产品入口 |
| 上传与解析 | 文稿析星 | 导入资料 |
| 复习与编码 | 光纹校准 | 确认知识与记忆编码 |
| 记忆宫殿 | 印忆殿 | 空间路线 |
| 训练与验证 | 知境试炼 | 主动回忆训练 |
| 学习报告 | 成长星图 | 学习数据报告 |
| 社区路线 | 共建星河 | 路线社区 |
| 质量门禁 | 知衡门禁 | 跨工具检查 |

## 4. 英文命名方向

- Knowledge Route
- Memory Encoding
- Local Asset Vault
- Recall Trial
- Growth Atlas
- Community Route Trust
- Quality Gate

## 5. 品牌禁区

```text
不要把 Lumina AI 改成纯中文品牌。
不要所有按钮都用诗意词。
不要让社区风险提示变得含糊。
不要把记忆宫殿做成游戏标题党。
```
